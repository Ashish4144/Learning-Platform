# Steganography Tools

| Tool | Use | Link |
|---|---|---|
| steghide | Hide/extract data in image & audio files | https://github.com/StefanoDeVuono/steghide |
| zsteg | Detect hidden data in PNG/BMP files | https://github.com/zed-0xff/zsteg |
| stegsolve | Java GUI for bit-plane/color analysis of images | https://github.com/eugenekolo/sec-tools/tree/master/stego/stegsolve |
| StegCracker | Brute-force steghide passphrases | https://github.com/Paradoxis/StegCracker |
| binwalk | Extract embedded/hidden files from any file | https://github.com/ReFirmLabs/binwalk |
| exiftool | Check metadata for hidden strings | https://exiftool.org/ |
| Audacity | Inspect audio files for spectrogram-hidden messages | https://www.audacityteam.org/ |
| SilentEye | Cross-platform image/audio steganography tool | https://github.com/achorein/silenteye |
| OpenStego | Open-source steganography tool with watermarking | https://www.openstego.com/ |
