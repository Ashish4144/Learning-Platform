# Mobile (Android / iOS) Tools

| Tool | Use | Link |
|---|---|---|
| jadx | Decompile Android APKs to readable Java | https://github.com/skylot/jadx |
| apktool | Decode/rebuild Android APK resources & smali | https://github.com/iBotPeaches/Apktool |
| MobSF | Automated mobile app security scanner (Android/iOS) | https://github.com/MobSF/Mobile-Security-Framework-MobSF |
| Frida | Dynamic instrumentation for runtime hooking | https://github.com/frida/frida |
| objection | Runtime mobile exploration built on Frida | https://github.com/sensepost/objection |
| Genymotion / Android Studio Emulator | Run APKs in a sandboxed virtual device | https://www.genymotion.com/ |
| ADB (Android Debug Bridge) | Interact with Android devices/emulators from CLI | https://developer.android.com/tools/adb |
| Cycript / Frida (iOS) | Runtime manipulation of iOS apps | https://github.com/frida/frida |
