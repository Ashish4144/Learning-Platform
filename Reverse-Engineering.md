# Reverse Engineering Tools

| Tool | Use | Link |
|---|---|---|
| Ghidra | Free NSA-built disassembler/decompiler | https://github.com/NationalSecurityAgency/ghidra |
| IDA Free/Pro | Industry-standard disassembler | https://hex-rays.com/ida-free/ |
| Binary Ninja | Modern disassembler with scripting API | https://binary.ninja/ |
| radare2 | Open-source reverse engineering framework | https://github.com/radareorg/radare2 |
| Cutter | GUI front-end for radare2 | https://github.com/rizinorg/cutter |
| x64dbg | Windows binary debugger | https://github.com/x64dbg/x64dbg |
| dnSpy / ILSpy | Decompile & debug .NET binaries | https://github.com/dnSpyEx/dnSpy |
| JD-GUI / CFR | Decompile Java class files | https://github.com/java-decompiler/jd-gui |
| Detect It Easy (DIE) | Identify packers/compilers/file types | https://github.com/horsicq/Detect-It-Easy |
| UPX | Unpack UPX-packed binaries | https://github.com/upx/upx |
| strings / strace / ltrace | Quick static/dynamic inspection on Linux | built into most distros |
