# Forensics Tools

| Tool | Use | Link |
|---|---|---|
| Autopsy | Full disk-image forensic analysis GUI | https://www.autopsy.com/ |
| Sleuth Kit | Command-line disk/filesystem forensics | https://github.com/sleuthkit/sleuthkit |
| Volatility 3 | Memory (RAM dump) forensics framework | https://github.com/volatilityfoundation/volatility3 |
| Wireshark | Packet capture analysis (.pcap) | https://www.wireshark.org/ |
| NetworkMiner | Passive network forensics from pcaps | https://www.netresec.com/?page=NetworkMiner |
| binwalk | Extract embedded files/firmware from binaries | https://github.com/ReFirmLabs/binwalk |
| foremost / scalpel | File carving from raw disk images | https://github.com/korczis/foremost |
| exiftool | Read/edit file metadata (EXIF, etc.) | https://exiftool.org/ |
| bulk_extractor | Extract emails, URLs, credit card numbers from images | https://github.com/simsong/bulk_extractor |
| FTK Imager | Disk imaging and preview tool | https://www.exterro.com/ftk-imager |
| CyberChef | Quick data transformation/carving | https://gchq.github.io/CyberChef/ |
